# PROJETO-FINAL-APC

Matrícula: 252014653\
Nome: Deivlim Tomaz Sousa de Azevedo

Este projeto é uma aplicação em C desenvolvida para simular as funcionalidades básicas de um portal acadêmico. O objetivo é gerenciar o cadastro de alunos e os processos de matrícula em matérias e trancamento de
matérias.

O programa permite que os estudantes se cadastrem, e recebam uma matrícula e através dessa matrícula gerada eles conseguem fazer o login no portal. Já dentro do portal, é possível se matricular nas
matérias disponíveis. Os estudantes poderão se matricular em ATÉ 7 matérias.

Além da matrícula, será possível visualizar o quanto de crédito e de horas os alunos irão receber ao término do semestre/matéria. Caso tranque alguma matéria, a mesma será excluída do cálculo de créditos.
